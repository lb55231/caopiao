# 移动端底部导航栏更新说明

## 更新内容

本次更新为移动端前台添加了完整的 5 个底部导航标签功能，参考了 wap.com 的 UI 和功能设计。

## 新增功能

### 1. 底部导航栏布局
- **文件**: `src/layout/TabbarLayout.vue`
- **功能**: 统一的底部导航栏布局组件
- **标签**:
  - 首页 (wap-home-o)
  - 订单 (orders-o)
  - 商家列表 (shop-o)
  - 活动 (gift-o)
  - 我的 (user-o)

### 2. 订单页面
- **文件**: `src/views/orders/Index.vue`
- **功能**:
  - 全部/待开奖/已中奖/未中奖 标签切换
  - 下拉刷新
  - 上拉加载更多
  - 订单列表展示
  - 订单状态标识
  - 空状态提示

### 3. 商家列表页面
- **文件**: `src/views/merchants/Index.vue`
- **功能**:
  - 搜索功能
  - 分类筛选（全部/快3/时时彩/11选5/PK10）
  - 商家卡片展示（logo、名称、描述、状态）
  - 商家统计信息（每日期数、开奖频率、营业时间）
  - 营业状态标识
  - 立即下注按钮

### 4. 活动中心页面
- **文件**: `src/views/activity/Index.vue`
- **功能**:
  - 轮播广告位
  - 活动分类（全部/进行中/即将开始/已结束）
  - 活动卡片展示（封面、标题、描述、时间、奖励）
  - 活动状态标识
  - 立即参与按钮

## 修改的文件

### 1. 路由配置
- **文件**: `src/router/index.js`
- **修改**:
  - 引入 TabbarLayout 布局组件
  - 将主要页面包裹在布局组件中
  - 添加新页面路由配置

### 2. 首页
- **文件**: `src/views/home/Index.vue`
- **修改**:
  - 移除内置的底部导航栏
  - 移除 `active` 状态变量
  - 调整样式（移除 padding-bottom）

### 3. App.vue
- **文件**: `src/App.vue`
- **修改**:
  - 简化 router-view 结构
  - 将 keep-alive 逻辑移至布局组件

## 页面路由

| 路径 | 页面 | 认证 | 缓存 |
|------|------|------|------|
| `/home` | 首页 | 否 | 是 |
| `/orders` | 订单 | 是 | 否 |
| `/merchants` | 商家列表 | 否 | 是 |
| `/activity` | 活动中心 | 否 | 是 |
| `/user` | 我的 | 是 | 否 |

## UI 设计特点

### 1. 统一的设计风格
- 圆角卡片设计
- 柔和的阴影效果
- 主题色：#ff6034（橙红色）
- 辅助色：#07c160（绿色）

### 2. 交互体验
- 下拉刷新
- 上拉加载更多
- 点击反馈动画
- 空状态提示
- 加载状态提示

### 3. 响应式布局
- 自适应移动端屏幕
- 栅格布局
- 弹性盒子布局

## 技术栈

- **Vue 3**: 组合式 API
- **Vue Router**: 路由管理
- **Vant 4**: UI 组件库
- **Sass**: CSS 预处理器

## 使用说明

### 安装依赖
```bash
cd vue/mobile
npm install
```

### 开发环境运行
```bash
npm run dev
```

### 生产环境打包
```bash
npm run build
```

## 注意事项

1. **图片资源**: 部分图片资源（如活动封面、轮播图）需要替换为实际的图片
2. **API 接口**: 订单和活动页面的数据接口需要后端配合开发
3. **权限控制**: 已添加路由守卫，未登录用户访问需要认证的页面会跳转到登录页
4. **兼容性**: 建议在 iOS Safari 和 Android Chrome 浏览器中测试
